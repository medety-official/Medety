# Medety (メデティ) 🌱
**「暗記」を「発見」に変える、語源学習アプリ**

Medetyは、膨大な医学単語を語源に分解し、パズルのように組み合わせて理解するための学習ツールです。

## ✨ 特徴
- **語源から学ぶ**: 単語を最小単位に分解し、意味の由来を直感的に理解。
- **分野別学習**: 循環器、解剖学など、医学部の試験範囲に合わせたカテゴリ分け。
- **語源ネットワーク**: 同じ語源を持つ単語を「串刺し」にして一気に習得。



1. **学ぶ**: カテゴリや語源を選んで学習を開始します。
2. **発見**: 知らない単語に出会ったら、その「パーツ」に注目してみてください。
3. **フィードバック**: 誤字や機能要望は、googleフォームからお知らせください。

## 🛠️ 開発の背景
現役の医学生として、膨大な医学用語の暗記に直面する中で、「語源を知ることで、暗記はもっと楽しく、効率的になるはずだ」という確信から開発をスタートしました。
